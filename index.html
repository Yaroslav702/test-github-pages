Що таке список

Список - упорядкований змінюваний контейнер даних. Списки не диктують розробнику тип даних, який можна помістити в цей контейнер, і можуть містити будь-які типи даних у будь-якому зручному порядку.
Список - це послідовність значень (подібна до масиву в інших мовах програмування, але більш універсальна). Значення у списку називаються елементами.

Властивості списків

Списки Python мають такі важливі властивості:
  • Списки впорядковані – списки запам’ятовують порядок вставлених елементів.
  • Доступ за індексом – доступ до елементів у списку можна отримати за допомогою індексу.
  • Списки можуть містити будь-які об’єкти – це можуть бути числа, рядки, кортеж.
  • Списки можна змінювати – ви можете змінювати список на місці, додавати нові елементи, а також видаляти чи оновлювати наявні елементи.

Створення списків

  • Для створення порожнього списку є два способи:
my_list = list()
empty_list = []
  • Створення заповненого списку:
not_empty = [1, 2, 'user']
Інші типи даних можна перетворювати в списки за допомогою конструктора list() мови Python.

Упорядковані контейнери

Доступ за індексом

Під упорядкованістю слід розуміти властивість контейнера зберігати порядок елементів під час роботи з ним.
Видаляючи елемент, додаючи новий, роблячи вставку/видалення з кінця/початку/середини, ви гарантуєте, що всі елементи, крім тих, яких безпосередньо торкнулася операція, зберігають порядок.
Найкориснішою властивістю впорядкованості є можливість доступу до елементів контейнера за індексом цього елемента в контейнері.
Індекс у Python починається з 0, як і в більшості мов програмування.
індексація елементів
У Python синтаксис доступу за індексом виглядає так:
some_iterable = ["a", "b", "c"]
first_letter = some_iterable[0]
middle_one = some_iterable[1]
last_letter = some_iterable[2]
ПОЯСНЕННЯ:
  • У першому рядку ми створили список із трьох перших літер англійського алфавіту.
  • У другому рядку ми зберегли у змінну first_letter літеру "a", перший елемент some_iterable.
  • Третій рядок - це звернення до другого елемента some_iterable, його індекс дорівнює 1 - це буква "b", і ми зберігаємо її в middle_one.
  • Четвертий рядок - це звернення до останнього елемента some_iterable, літери "c". Ми збережемо її в last_letter, і її індекс дорівнює 2.

Доступ до елементів вкладених списків

Аналогійно можна отримати доступ до окремих елементів у вкладеному списку за допомогою декількох індексів.
Перший індекс визначає, який список використовувати, а другий вказує на значення в цьому списку.
Індекси для елементів у вкладеному списку показано нижче:
індекси елементів вкладеного списку
Приклад
Результат
L = ['a', 'b', ['cc', 'dd', ['eee', 'fff']], 'g', 'h']
​
print(L[2][2])
# Prints ['eee', 'fff']
​
print(L[2][2][0])
# Prints eee

Використання методів об'єктів

Об'єкт у програмуванні - деяка сутність у цифровому просторі, що володіє певним станом і поведінкою, має певні властивості (атрибути) та операції над ними (методи).
Доступ до методів об'єктів у Python синтаксично відбувається за допомогою символу крапки після імені об'єкта і вказівки імені методу або атрибута, до якого потрібно отримати доступ.
Приклад:
numbers = ['a', 'b']
numbers.append('c')
print(numbers) # ['a', 'b', 'c']
ПОЯСНЕННЯ: У цьому прикладі ми використовували метод append, який є у списків. Він є у списку numbers. Цей метод додає новий елемент у кінець списку. Як аргумент цей метод отримує елемент, який треба додати до списку. Аргументи вказуються в дужках.
Якщо метод не вимагає аргументів (наприклад, метод clear), то дужки будуть порожніми:
num = [1, 2]
num.clear()
print(num) # []

Методи списків

  • Додавання елемента в кінець списку: my_list.append(element).
Приклад:
chars = ['a', 'b']
chars.append('c')
print(chars) # ['a', 'b', 'c']
  • Видалення елемента зі списку (призведе до помилки, якщо такого елемента немає в списку).
Приклад:
chars = ['a', 'b']
chars.remove('b')
print(chars) # ['a']
Але майте на увазі, що якщо в списку наявно більше одного екземпляра цього елемента, то за допомогою методу можна видалити тільки перший екземпляр.
  • Повернути і-й елемент й видалити його зі списку і_element = my_list.pop(i). За замовченням і = -1.
Приклад:
chars = ['a', 'b']
last = chars.pop(1)
print(chars) # ['a']
print(last) # 'b'
  • Якщо видалене значення не потрібне, використовуйте інструкцію del.
Приклад:
chars = ['a', 'b']
del chars[1]
print(chars) # ['a']
  • Розширити список a_list елементами з b_list: a_list.extend(b_list).
Приклад:
chars = ['a', 'b']
numbers = [1, 2]
chars.extend(numbers)
print(chars) # ['a', 'b', 1, 2]
  • Вставити x на позицію з індексом i: my_list.insert(i, x).
Приклад:
chars = ["a", "b"]
chars.insert(1, "c")
print(chars) # ['a', 'c', 'b']
  • Очистити список: my_list.clear().
Приклад:
chars = ['a', 'b']
last = chars.clear() print(chars) # []
  • Знайти індекс першого елемента у списку, що дорівнює x: index = my_list.index(x).
Приклад:
chars = ['a', 'b', 'c', 'd']
c_ind = chars.index('c') print(c_ind) #2
  • Повернути кількість елементів, рівних x, до списку: x_number = my_list.count(x).
Приклад:
chars = ['a', 'b', 'c', 'a']
a_count = chars.count('a')
print(a_count) # 2
  • Впорядкувати список за зростанням: my_list.sort(key=None, reverse=False).
Приклад:
chars = ['z', 'a', 'b']
chars.sort()
print(chars) # ['a', 'b', 'z']
  • Змінити порядок елементів у списку на зворотний: my_list.reverse().
Приклад:
chars = ['a', 'b']
chars.reverse()
print(chars) # ['b', 'a']
  • Повернути копію списку: copy_of_my_list = my_list.copy().
Приклад:
chars = ['a', 'b']
chars_copy = chars.copy()
chars == chars_copy # True
  • Знайти довжину списку Щоб знайти кількість елементів у списку, використовується метод len().
Приклад:
chars = ['a', 'b']
print(len(chars)) # 2
  • Перевірка наявності елемента в списку. Для визначення наявності чи відсутності значення в списку можна використовувати оператори in та not in з інструкцією if.
Приклад:
chars = ['a', 'b']
if 'a' in chars:
print('yes')
if 'www' not in chars:
print('no')
  • Ітерація за списком. Найпоширеніший спосіб ітерації за списком - це цикл for.
Приклад:
chars = ['a', 'b', 'c']
for item in chars:
print(item)

Практичне завдання

Уявімо, що ми розробляємо інформаційну систему продуктового магазину, яка має виконувати наступні функції:
  • Показ усіх наявних товарів (виведення елементів списку),
  • Постачання нових товарів (додавання елементів, об’єднання списків),
  • Продаж товарів (пошук елемента списку і видалення його),
  • Заміна проданих продуктів на нові (вставка елемента),
  • Виведення списку проданих продуктів за день,
  • Показати історію продажів (виведення списку в зворотному порядку).
